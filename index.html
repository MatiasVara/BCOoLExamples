BCOoLExamples

The project contains examples of operators specified by using BCOoL. After compiled, operators can be used to generate the model of coordination between models. Examples of operators are: synchronized product between Timed Finite State Machine (TFSM), synchronized product between TFSM and Activity diagrams and more. For further information, go to http://timesquare.inria.fr/BCOoL

BCOoLExamples
